# Dot-art

A collection of dotart logos for use with [fastfetch](https://github.com/fastfetch-cli/fastfetch).

## About

This folder contains ASCII and Unicode art logos designed to be used with fastfetch, a fast and highly customizable system information tool. The dotart files can be used to personalize your system info output.

## Usage

1. Copy your preferred dotart file from this directory.
2. Place it in the directory you want and reference it in fastfetch config, commonly `~/.config/fastfetch/config.jsonc`.
3. Run fastfetch to see your custom logo

```bash
fastfetch
```
